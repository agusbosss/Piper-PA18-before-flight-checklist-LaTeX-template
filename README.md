# Piper PA-18 Before Flight Checklist (LaTeX Template)

LaTeX template to generate a clean, printable checklist PDF for a Piper PA-18 (or similar).

## Preview
- PDF: `Pipper_PA18_before_flight_checklist.pdf`

## Build
Compile with:
- Overleaf (recommended), or
- `latexmk -pdf main.tex`
